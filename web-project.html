Homework 3: How websites work?

The website you are viewing was created based on the instructions we received for our "Homework 3: How Websites Work?" assignment for the Principles of Information Systems course. This assignment required us to collaboratively build a team website using web development languages: HTML for structure, CSS for styling, and JavaScript for dynamism. Our task was not just to create a functional website but to ensure it was informative, rich in content, and engaging.We used our skills acquired from Codecademy courses and worked in a GitHub Classroom repository. This required us to familiarize ourselves with tools like Git for version control and Visual Studio Code for development. We made sure to follow the instructions and created an original website through close teamwork. In addition to technical knowledge, this experience improved our understanding of collaboration and coordination as we had to avoid merge conflicts while using Git. These skills will surely be put to better use in the future .

The Process

The creation process began with sharing the tasks among members. Nurlan designed the "Team Members" page, Samira developed the "Hour of Code" section, Yusif did the "Home Page" and the "Code Art" Bonus task, and Ilham made the "Web Project" page and certain dynamic elements. JavaScript was used for dynamic elements. For example, color theme button, greeting screen, and the code art project were all created using JavaScript. Communication played a key role in our process. We ensured that all members were using Git and properly configured their IDEs (Microsoft Visual Studio Code). After completing individual tasks, we published the website using GitHub as a bonus task. The project not only enhanced our technical skills but also highlighted the importance of teamwork in confronting challenges.
