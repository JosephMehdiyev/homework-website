Homework 3: How websites work?

The website you are viewing at the moment was created based on the instructions that we received for the "Homework 3: How websites work?" assignment of our "Principles of Information Systems" course. We used the knowledge that we gained from our Codeacademy courses on HTML, CSS and JS. We were instructed to make the website informative, rich and interactive.The assignment was shared amongst each team member with every member working on their individual tasks.This experience enhanced our teamwork skills and taught us how to use Git which is a crucial application for developers. These skills will definitely reflect better results in the future.

The Process

The assignment was shared amongst each team member with Nurlan working on the "Team members", Samira on the "Hour of Code", Yusif on the "Home page" and Bonus tasks, lastly, Ilham on the "Web project" page. We used HTML for the fundamental structure, CSS for styling and lastly JS for certain dynamic elements such as the color theme button, the greeting screen and the code art project which was a bonus task. Communication was a crucial aspect in the process. We had to make sure that every member had correctly tuned their IDEs (MS VSC) i.e., were not having trouble committing and pulling or creating merge conflicts etc. Once all were done, we published the website on GitHub.
